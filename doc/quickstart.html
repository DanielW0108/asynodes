Nodes (Quickstart)

Node creation from strings

  Node n = Node ("foo");
  n.stamp();

Make a node containing only the text foo, and draw it onto the current picture centred at (0, 0).

  Node n = Node ("foo", pos=(2, 3));
  n.stamp();
  n.moveTo(8, 8);
  n.stamp();

Make the same node, draw it once at (2, 3), and then draw it again at (8, 8).

  Node n = Node ("foo", draw = black, fill = rgb(230, 230, 355));

Make a node containing the text foo surrounded by a black box with a 3mm margin, with a light blue background.

  Node n = Node ("foo", draw = black, fill = rgb(230, 230, 355), xmargin = 5mm, ymargin = 3mm, shape = roundedbox);

Make a similar node, but with a 5mm x margin, 3mm y margin, and a rounded box.

Using pictures to make nodes

  picture pic;
  filldraw(pic, unitcircle, draw=black, fill=red);

  Node n = Node(pic, draw=black);

Make node containing a red circle with a black outline, surrounded by a black box with a 3mm margin.

  picture pic;
  filldraw(pic, unitcircle, draw=black, fill=red);

  Node n = Node(pic, draw=black, midpoint = (1mm, 0));

Make a similar node, but make the “centre point” of the node 1mm to the right. This means that n.anchor(south) will not be precisely in the middle of the southernmost edge of the box, it will instead be shifted across by 1mm.

Positioning and connecting nodes

  Node a = Node("foo");
  Node b = Node("bar");
  Path p = b.moveNextTo(a, (0, 1cm));

  a.stamp(); b.stamp(); draw(p);

Make two plain nodes, place the second 1cm to the right of the first, and draw a line connecting them.
